Alert

Alert is a notification type which is fixed into the middle of the screen. It has a content box which has a 425px max-width.

Modal

Modal is a notification type which is full-screen-sized. It can be filled with any content.

Popup

Popup should be a notification type which opens by sliding from the bottom of the page. You are supposed to write code for the popup notification including Javascript, CSS and HTML. Please check readMe.txt file before doing it.
